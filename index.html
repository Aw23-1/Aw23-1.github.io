大家好！这是我的第一个通过 GitHub Pages 托管的静态博客。 本页面展示了绝对路径和相对路径两种图片引用方式， 并已集成评论区功能（基于Giscus）。

🌐 一、绝对路径图片

📍 绝对路径：https://picsum.photos/id/104/800/400
💡 绝对路径解释：
以 https:// 或 http:// 开头的完整URL，或者以根目录 / 开头的路径。 这里使用的是网络图片服务 picsum.photos 的完整链接，无论页面在哪个服务器都能正常加载。

📁 二、相对路径图片

📍 相对路径：image/local.jpg
💡 相对路径解释：
相对于当前 index.html 文件所在目录。
当前文件位置：/index.html → 图片位置：/image/local.jpg
✅ 部署后，仓库里必须有 image/local.jpg 这张图片才会正常显示。

💬 三、关于评论区

由于 GitHub Pages 只能托管静态文件，无法直接运行后端代码。
本页面借助 Giscus 实现评论功能，所有评论都会存储在你的 GitHub 仓库的 Discussions（讨论区）中，完全免费且无广告。
